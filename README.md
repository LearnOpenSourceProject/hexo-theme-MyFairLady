### Installation

    $ git clone https://github.com/yzzting/hexo-theme-MyFairLady.git

Then update your blog's main **_config.yml** to set the theme to MyFairLady    

    # Extensions
    ## Plugins: http://hexo.io/plugins/
    ## Themes: http://hexo.io/themes/
    theme: MyFairLady
